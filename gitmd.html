GitMD

あなたのノート。あなたのリポジトリ。あなたのルール。

AndroidLinuxmacOSWindows

Android とデスクトップ（Linux / macOS / Windows）両対応の Markdown ノートアプリ。ノートはプレーンな .md ファイルとして、 あなたが選んだ場所— Git リポジトリ、Google Drive、Dropbox、Box、または端末ローカル — に保存されます。 ベンダーロックインなし。あなたが管理できないクラウドもありません。

アプリを入手

GitMD を選ぶ理由

いつでもプレーンな Markdown

ノートはディスク上の .md ファイルそのもの。どのエディタでも、いつでも、どの端末でも開けます。専用フォーマットなし、エクスポート作業も不要。

Git がファーストクラスのバックエンド

GitHub・GitLab・Codeberg・Bitbucket・Gitea・セルフホストの任意リポジトリをクローン。commit / push / pull に対応し、フル版履歴が残ります — それが Git だからです。

すでに信頼しているストレージ

Git を使いたくない？ Google Drive・Dropbox・Box・ローカルのみから選べます。同じノートを、好きな同期方法で。

WikiLink とバックリンク

[[ページ名]] リンクがネイティブに動作。タップで移動でき、存在しないページは自動作成。バックリンクパネルで、現在のノートを参照している箇所が分かります。

必要なときだけ AI

Gemini・OpenAI・Anthropic のキーを持ち込んで利用。使わなければ 100% オフラインのまま。キーは端末内で暗号化されます。

ジャーナルモード

ワンタップで journal/YYYY-MM-DD.md を開きます。Android の共有シートから任意のアプリの内容を追記。気負わず日々のノートを。

  • ✔ Git — GitHub OAuth、GitLab OAuth、個人アクセストークン（PAT）、SSH キー。Codeberg / Bitbucket / Gitea / セルフホスト Forgejo はすべて動作確認済み。

  • ✔ Box — OAuth でフォルダ選択。

  • ✔ Google Gemini — gemini-2.5-flash
  • ✔ OpenAI — gpt-4o-mini
  • ✔ Anthropic — claude-haiku-4-5

キーは Android の EncryptedSharedPreferences（Keystore ベース）で保存されます。AI を完全に無効化することも可能 — プロバイダ未設定でもアプリは動作します。

プライバシー

あなたのノートが、選んだストレージ（Git リモート / Drive / 端末）の外に出ることはありません。 トークンは端末内で暗号化されます。匿名の利用イベントのみ Firebase Analytics に送信されます （個人情報なし、広告 ID なし）。アカウントもプロフィール紐付けもありません。

GitMD

Your notes. Your repo. Your rules.

AndroidLinuxmacOSWindows

A Markdown notes app for Android and desktop (Linux / macOS / Windows). Notes live as plain .md files in your Git repository — or Google Drive, Dropbox, Box, or just your filesystem. No vendor lock-in. No cloud you don’t control.

Get the app

Why GitMD

Plain Markdown, always

Your notes are .md files on disk. Open them with any editor, any time, on any device. No proprietary format. No export step.

Git is a first-class backend

Clone any GitHub, GitLab, Codeberg, Bitbucket, Gitea, or self-hosted repo. Commit, push, pull. Full version history because that’s just how Git works.

Storage you already trust

Don’t want Git? Pick Google Drive, Dropbox, Box, or local-only. Same notes, your choice of sync.

Wiki links & backlinks

[[Page]] links work natively. Tap to navigate. Missing pages get auto-created. Backlinks panel shows what references the current note.

AI when you want it

Bring your own key for Gemini, OpenAI, or Anthropic. Stay 100% offline if you don’t. Your key is encrypted on-device.

Journal mode

One tap opens journal/YYYY-MM-DD.md. Append from any app via the Android share sheet. Daily notes without ceremony.

Storage backends

  • ✔ Git — GitHub OAuth, GitLab OAuth, Personal Access Token, SSH key. Codeberg / Bitbucket / Gitea / self-hosted Forgejo all confirmed working.
  • ✔ Local filesystem — SAF-backed folder on your device, no cloud at all.
  • ✔ Google Drive — OAuth + Drive Picker for folder selection.
  • ✔ Dropbox — OAuth folder selection.
  • ✔ Box — OAuth folder selection.

AI providers (bring your own key)

  • ✔ Google Gemini — gemini-2.5-flash
  • ✔ OpenAI — gpt-4o-mini
  • ✔ Anthropic — claude-haiku-4-5

Keys are stored with Android EncryptedSharedPreferences (Keystore-backed). Disable AI entirely if you prefer — the app works without any provider configured.

Privacy

Your notes never leave your storage backend (Git remote / Drive / device). Tokens are encrypted on-device. Anonymous usage events are sent to Firebase Analytics (no PII, no advertising ID). No accounts, no profile linking.
